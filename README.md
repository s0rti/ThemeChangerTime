# ThemeChangerTime
(change theme script) light: 8:00-20:00 dark: 20:00-8:00 (you can change time if you want)

# How to install?
`download themeChangeTimeMacOs (MacOs version) or themeChangeTimeWin (Windows version)`

Open your terminal and enter the path to your script file, for example, `cd path to your script file/`
